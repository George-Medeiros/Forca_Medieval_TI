# Medieval TI - Jogo da Forca ⚔️🛡️

Este é um projeto de jogo da forca com temática medieval desenvolvido para a disciplina de Algoritmos e Programação da **UNINTER**. O jogo utiliza conceitos de TI como palavras-chave, unindo o aprendizado técnico ao entretenimento.

## 👤 Desenvolvedor
* **Nome:** George Medeiros

## 🎮 O Jogo
O objetivo é adivinhar a palavra secreta relacionada ao mundo da Tecnologia da Informação antes que o personagem seja totalmente enforcado. Cada erro revela uma parte do corpo no cadafalso.

### Funcionalidades:
- **Interface Gráfica:** Desenvolvida com a biblioteca Pygame.
- **Dicas Dinâmicas:** Sistema de dicas em alto contraste (Dourado) para melhor legibilidade.
- **Ambientação:** Trilha sonora medieval e artes customizadas.
- **Portabilidade:** Versão compilada em executável para Windows.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Biblioteca Gráfica:** Pygame
- **Compilação:** PyInstaller
